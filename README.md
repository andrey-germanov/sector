# Сектор - Лендинг сообщества

Современный лендинг для сообщества Сектор с темным дизайном, анимациями и интерактивностью.

## Технологии

- React 18
- TypeScript
- Vite
- Framer Motion (анимации)
- Firebase (для обратной связи - будет добавлено позже)

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Функционал

### Обратная связь
- Модальное окно с формой для отправки обратной связи
- Ограничение отправки: 1 раз в минуту (через localStorage)
- Данные сохраняются в Firebase Firestore

### Страница данных
- Доступна по адресу `/data`
- Защищена паролем: `Sektor2024!Data`
- Отображает все полученные сообщения обратной связи в реальном времени
- Автоматическое обновление при новых сообщениях

## Firebase

Проект настроен для работы с Firebase Firestore. Данные сохраняются в коллекцию `feedback` со следующими полями:
- `telegramNick` - ник пользователя в Telegram
- `message` - текст сообщения
- `timestamp` - время создания (serverTimestamp)
- `createdAt` - ISO строка времени создания
